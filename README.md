# Портфолио проектов — Res Space

Статический сайт-портфолио в фирменном стиле Res Space. Показывает проекты,
раскрывает почему каждый было сложно сделать, стек/подходы и галерею скринов.

## Как открыть

Просто открыть `index.html` в браузере (или запустить любой静ический сервер
из папки `portfolio/`). Никакой сборки не нужно — чистый HTML/CSS/JS.

```bash
cd ~/portfolio
python3 -m http.server 8080
# → http://localhost:8080
```

## Структура

```
portfolio/
├── index.html            # Главная — обзор всех проектов
├── device-rental.html    # Device Rental Service (флагман)
├── cs-manager.html       # CS Server Manager
├── booking-notifier.html # booking_notifer
├── reviews-bot.html      # Бот-сборщик отзывов
├── css/style.css
├── js/main.js
└── assets/               # ← сюда кладёшь скрины
```

## Как добавить скрины

Каждый скрин на сайте — это **подписанный плейсхолдер** с точным путём файла.
Чтобы плейсхолдер заменился реальной картинкой, просто положи файл по
указанному ниже пути (имя должно совпадать точь-в-точь). Страницу не трогать —
JS сам подхватит картинку при загрузке.

Формат — PNG или JPG (переименуй в `.png`). Оптимально 16:10 для десктоп-
скринов, 9:16 для скринов телефона (Telegram-бот).
